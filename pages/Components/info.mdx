# Componente Info

El componente `Info` es un componente que muestra información general sobre Colab Tools. Este componente incluye varios subcomponentos y elementos de UI para crear una experiencia de usuario atractiva.

## Estructura del Componente

El componente `Info` se compone de los siguientes elementos:

- `section`: Este es el contenedor principal del componente.
- `div`: Estos son los contenedores secundarios que proporcionan estructura al componente.
- `Button`: Este es un botón que redirige al usuario a la sección de aplicaciones.
- `Link`: Este es un enlace que redirige al usuario a la sección de aplicaciones.
- `p`: Este es un párrafo que proporciona información sobre Colab Tools.
