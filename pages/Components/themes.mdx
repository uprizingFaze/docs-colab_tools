# Componente ThemeProvider

El componente `ThemeProvider` es un componente de orden superior que proporciona el tema actual a todos los componentes de su árbol de componentes. Este componente utiliza el `ThemeProvider` de `next-themes` para proporcionar esta funcionalidad.

## Estructura del Componente

El componente `ThemeProvider` se compone de los siguientes elementos:

- `NextThemesProvider`: Este es el proveedor de temas de `next-themes`. Proporciona el tema actual a todos los componentes de su árbol de componentes.

## Propiedades

El componente `ThemeProvider` acepta las siguientes propiedades:

- `children`: Los componentes hijos que se renderizarán dentro del `ThemeProvider`.
- `...props`: Cualquier otra propiedad que se pasará al `NextThemesProvider`.

## Código

Aquí está el código del componente `ThemeProvider`:

``` tsx
"use client";

import * as React from "react";
import { ThemeProvider as NextThemesProvider } from "next-themes";
import { type ThemeProviderProps } from "next-themes/dist/types";

export function ThemeProvider({ children, ...props }: ThemeProviderProps) {
  return <NextThemesProvider {...props}>{children}</NextThemesProvider>;
}
```
