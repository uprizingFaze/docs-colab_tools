# Guía de Instalación del Proyecto Colab_Tools

Este proyecto está construido con Next.js. Sigue los siguientes pasos para instalar y ejecutar el proyecto en tu máquina local.

## Pre-requisitos

Antes de comenzar, asegúrate de tener lo siguiente instalado en tu máquina:

- Node.js: Necesitarás Node.js para ejecutar el código de JavaScript en tu máquina. Puedes descargarlo desde [aquí](https://nodejs.org/).
- Git: Necesitarás Git para clonar el repositorio. Puedes descargarlo desde [aquí](https://git-scm.com/).
- Yarn o npm: Estos son gestores de paquetes de Node.js. Puedes descargar Yarn desde [aquí](https://yarnpkg.com/) o npm viene preinstalado con Node.js.

## Clonar el Repositorio

Primero, clona el repositorio en tu máquina local utilizando Git. Abre una terminal y ejecuta el siguiente comando:

```bash
git clone https://github.com/uprizingFaze/colab_tools.git
```

## Instalar las Dependencias

Navega a la carpeta del proyecto y instala las dependencias utilizando Yarn o npm:

```bash
cd colab_tools
yarn install
```

o

```bash
cd colab_tools
npm install
```

## Ejecutar el Proyecto

Finalmente, puedes ejecutar el proyecto en tu máquina local utilizando el siguiente comando:

```bash
yarn dev
```

o

```bash
npm run dev
```

Ahora, deberías poder ver el proyecto ejecutándose en `http://localhost:3000` en tu navegador.
