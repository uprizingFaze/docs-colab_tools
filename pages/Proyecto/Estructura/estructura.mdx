# Estructura del Proyecto

El proyecto Colab Tools tiene la siguiente estructura de archivos y directorios:

```
└── 📁Colab_Tools
    └── README.md
    └── 📁app
        └── layout.tsx
        └── page.tsx
    └── 📁components
        └── 📁characters
            └── characters.tsx
        └── 📁hero
            └── hero.tsx
        └── 📁info
            └── info.tsx
        └── navbar.tsx
        └── 📁theme
            └── theme-provider.tsx
        └── 📁ui
            └── button.tsx
            └── dropdown-menu.tsx
            └── navbar-menu.tsx
    └── 📁lib
        └── utils.ts
    └── package.json
    └── 📁public
        └── 📁cursors
            └── cursor_1.svg
            └── cursor_2.svg
        └── code.png
        └── grid-bg.jpg
    └── tailwind.config.ts
    └── tsconfig.json
    └── 📁utils
        └── cn.ts
```
Esta estructura de archivos te ayudará a entender mejor cómo está organizado el proyecto.
