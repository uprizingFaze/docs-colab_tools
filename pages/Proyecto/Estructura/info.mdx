# Estructura del Proyecto Colab_Tools

El proyecto Colab Tools tiene la siguiente estructura de archivos y directorios:

## 📁app

Esta carpeta contiene los archivos de configuración y estilo global de la aplicación.

- `layout.tsx`: Define el diseño principal de la aplicación.
- `page.tsx`: Define la estructura de las páginas de la aplicación.

## 📁components

Esta carpeta contiene todos los componentes de React utilizados en la aplicación.

- 📁characters: Contiene los componentes relacionados con las Características.
- 📁hero: Contiene los componentes del hero de la aplicación.
- 📁info: Contiene los componentes de información.
- `navbar.tsx`: Define la barra de navegación de la aplicación.
- 📁theme: Contiene los componentes relacionados con el tema de la aplicación.
- 📁ui: Contiene los componentes de la interfaz de usuario.

## 📁lib

Esta carpeta contiene utilidades y funciones auxiliares.

- `utils.ts`: Contiene funciones auxiliares utilizadas en toda la aplicación.

## 📁public

Esta carpeta contiene los archivos estáticos públicos como imágenes y fuentes.

- 📁cursors: Contiene los archivos de cursor personalizados.
- `code.png`: Una imagen utilizada en la aplicación.

## 📁utils

Esta carpeta contiene utilidades y funciones auxiliares.

- `cn.ts`: Una función auxiliar para la concatenación de nombres de clases.

## Archivos de Configuración

- `package.json`: Contiene las dependencias del proyecto y los scripts de configuración.
- `tailwind.config.ts`: Configuración de Tailwind CSS, un marco de trabajo de CSS de utilidad primero.
- `tsconfig.json`: Configuración de TypeScript para el proyecto.
